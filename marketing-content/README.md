## Regles editoriales

- **Langue** : Francais courant, pas soutenu. Tutoiement.
- **Ton** : Expert bienveillant. Jamais condescendant.
- **Promotion** : JAMAIS de vente directe dans les posts groupes. Mentionner tloush max 1 fois en fin de post, comme une ressource utile.
- **Donnees** : Tous les chiffres sont bases sur les taux officiels 2025 (verifies dans le code source tloush).
- **Format** : Posts courts (150-300 mots). Pas de jargon technique sans explication.
